SASS Folder connect -

First go to your html files and add light-them.css or dark-theme.css in place of the css file that is linked in the head.

Then you have to click on watch sass taken from vscode editor.

After clicking, two automatic files will be created with light-theme.scss One is light-theme.css and the other is light-theme.css.map Then you have to understand that your sass file is connected.

Quick Changes color, font, & others -

First of all you have to connect sass folder in your html file. If you want to change the dark-sass or light-sass theme from the source folder, go to the helpers folder and open the variable.scss file.Then you can easily change any color, font etc.

Use the elements of elsewhere -

Each element, component, card of this template has been created separately so that you can use it elsewhere. Go inside the sass file and you will find all the components in the component, element, layout folders arranged separately.
